# FIROZA Website

A Next.js and Tailwind CSS storefront homepage for FIROZA.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

The project uses placeholder images from `picsum.photos`. Replace the image URLs in `lib/data.js` and `components/Hero.js` with final brand or product photography before launch.
